Mower
=======

Driving
----------

**Core features:**

- Easy to use wrapper functions for driving. For general cases
    - Drive forward
    - Drive backward
    - Turn X degree around center
    - Turn X degree around wheel
    - Drive arc (radius, left/right, forward/backward)
    - Stop
- Special driving functions for special cases
    - Set motors speeds + directions

- How long to drive:
    - Till next update
    - Distance (In a more abstract layer for way algorithm)
